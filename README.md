## Possus

Nessus report prescreener

### Prerequisites

* log4net
* Newtonsoft.Json
* RestSharp

## Usage
- Configure authentication
	- go to Auth/auth-example.json
	- change exampleurl, exampleusername, examplepassword
	- save the file and rename it to auth.json
-  Run the program
